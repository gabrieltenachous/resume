Descrição do Projeto

Projeto em Laravel, Jquery, IUGU (sistema de pagamento) em que os produtores postam suas fotos e vídeos podendo escolher se o conteúdo sera gratuito ou pago (com ou sem live) onde é implementado uma live ao vivo para os produtores fazerem seus conteúdos com os clientes pagos. O pagamento é usado a IUGU para fazer a compra dos conteúdos dos produtores em que o produtor recebe uma porcentagem dessa quantia para ele. O sistema é totalmente responsivo para mobile porque grande parte dos usuários são mobiles.
