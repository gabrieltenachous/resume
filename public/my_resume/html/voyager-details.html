Laravel Voyager

  • Objetivo: Sistema que cria pacotes de CRUD
  • Situação: Estudo (Github)
  • URL do Projeto: https://github.com/gabrieltenachous/VoyagerLaravel

Descrição do Projeto

O Voyager é um pacote de administração do Laravel que inclui operação BREAD (CRUD), um gerenciador de mídia, construtor de menus e muito mais. É possível nele fazer relacionamentos, criar seus próprios inputs, sistema de rotas, mídia, permissões etc. Tudo isso apenas no navegador.

Como usar:
php artisan voyager:admin admin@email.com –create
Esse é para ter uma acesso de usuário administrador por que o cadastro comum é só para usuários E vá em localhost/admin/login para fazer seu login!!
